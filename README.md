# Raito
A cli that aims to managing files

### Install
```
npm install -g raito
```
### Show all link
```
moon list
```
### Add a new folder link
```
moon add 'DESCRIPTION'
```
### Del a link
```
moon del [goid]
```
### Show the current path
```
moon pwd
```
### Clear all Link
```
moon clear
```
### Go to link
```
moon go [goid]
```
